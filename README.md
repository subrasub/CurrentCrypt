# CurrentCrypt

A web-app to view a list of various cryptocurriencies, and search for them as well.

|      URL      |    Function   |
| ------------- | ------------- |
| /search  | Search for any specific cryptocurrency  |
| /list  | List the summary of the top Cryptocurriencies  |
